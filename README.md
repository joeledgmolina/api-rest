# ApiRest

## Dependencies

`npm i express morgan`

`npm i nodemon -D`

`npm i lodash`

## To execute
inside the directory api-rest and run on console
`npm run dev `

## The RestApi have the following endpoints
1. Get all authors
2. Get all books with the author
3. Add an author
4. Add a book
5. Modify an author
6. Modify a book
7. Delete a author
8. Delete a book
